**Goal:** Show a speech-language pathologist that BlasterAI collapses *hours* of board-building into a sentence and a few seconds. This is the video that makes therapists lean in — it speaks directly to their daily pain.

## Notes for the editor
- **Type the goal on-screen, slowly enough to read.** The plain-English prompt *is* the wow — don't pre-fill it.
- If generation takes more than a few seconds live, keep the progress state but tighten it in the edit so it feels like "seconds."
- Show a **real generated result** — do not fake the scene. If the model produces something imperfect, that's fine and honest; pick a goal you've tested (the "frustration vs anger" prompt is the canonical one from the docs).
- The **share/portability** beat at 1:00 matters to therapists worried about lock-in — don't cut it.
- Landscape gives you room to show the preview and the pages; keep the admin chrome visible so it reads as a real tool, not a mockup.

## The one line if someone only reads the caption
> *Describe the goal in plain English → get a complete, editable, shareable AAC board in seconds.*
